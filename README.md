<h1> coinpayment </h1>

<p align="left"> CoinPayment 币种钱包地址生成</p>


## Installing

```shell
$ composer require jybtx/coinpayment
```

## Usage

```php
use CoinPayment;
CoinPayment::GetDepositAddress('currency');
```
